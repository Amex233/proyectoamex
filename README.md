# Proyecto AMEX – Scrum con React

Aplicación web prototipo para la gestión financiera del hogar,
desarrollada utilizando React y la metodología ágil Scrum.

## Tecnologías utilizadas
- React
- JavaScript
- react-router-dom
- GitHub

## Metodología Scrum
Se aplicó Scrum definiendo roles, artefactos y sprints.
Incluye Product Backlog, Sprint Backlog y Sprint Planning.

## Instalación
1. Clonar el repositorio
2. Ejecutar:
   npm install
   npm start
